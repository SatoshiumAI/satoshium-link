# Satoshium Shared Components

The Satoshium component layer provides shared interface elements used across multiple platform domains.

These components ensure that Satoshium operates as a coordinated ecosystem rather than a collection of unrelated sites.

They support consistency, scalability, and long-horizon architectural clarity.

---

## Purpose of Shared Components

Shared components allow Satoshium to:

- maintain consistent navigation across domains
- centralize platform identity elements
- reduce duplication between repositories
- support modular expansion of the ecosystem
- coordinate public interface behavior across layers

They function as connective infrastructure between platform surfaces.

---

## Core Component Systems

### Topbar Loader

The topbar loader dynamically inserts the correct navigation surface depending on page context.

Two primary variants exist:

- **Platform topbar** for primary platform pages
- **Universe topbar** for cross-domain navigation surfaces

This allows each domain to maintain its role while remaining visibly connected to the broader platform.

---

### Footer Loader

The footer loader provides consistent platform references across domains, including:

- mission framing
- legal links
- registry alignment surfaces
- platform navigation shortcuts

Centralizing the footer ensures updates propagate across the ecosystem automatically.

---

### Domain Layers Component

The domain layers component explains how each Satoshium domain fits within the overall architecture.

It helps users understand:

- platform structure
- domain responsibilities
- navigation relationships
- ecosystem layering logic

This component reinforces architectural transparency.

---

### Sayings Engine

The sayings engine rotates short platform reflections across pages.

These statements reinforce:

- platform philosophy
- coordination intent
- long-horizon development mindset
- decentralized intelligence principles

They provide continuity without repeating static messaging everywhere.

---

## Why Components Matter in Phase004

Phase004 establishes Satoshium as a distributed platform operating across multiple domains.

Shared components make this possible by:

- preserving visual coherence
- supporting modular infrastructure growth
- simplifying cross-domain navigation
- aligning public-facing surfaces
- maintaining architectural consistency

They act as lightweight coordination infrastructure for the interface layer.

---

Shared components help ensure the Satoshium ecosystem evolves as a unified platform rather than a fragmented collection of pages.
