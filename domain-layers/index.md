# Satoshium Domain Layers

Satoshium is not built as a single-site platform.

It is structured as a layered coordination system distributed across multiple domains. Each domain represents a distinct architectural surface within the broader Satoshium ecosystem.

Together, these domains form the public interface of decentralized intelligence infrastructure aligned with the Bitcoin standard.

---

## Purpose of the Domain Layer Model

The domain-layer structure allows Satoshium to:

- separate stable foundations from experiments
- distinguish infrastructure from interfaces
- organize knowledge independently from execution surfaces
- support long-horizon platform growth without centralizing control
- maintain clarity as the ecosystem expands

Each domain serves a specific role within the overall architecture.

---

## Core Platform Domains

### 🌐 satoshium.ai — Platform Layer

Primary public entry point to the Satoshium ecosystem.

Provides:

- platform introduction
- architecture overview
- registry access
- updates and progress visibility
- legal and governance surfaces

This domain explains what Satoshium is.

---

### 🛰️ satoshium.net — Infrastructure Layer

Documents systems, services, and coordination topology.

Provides:

- system registry alignment
- service definitions
- verification surfaces
- Signal Layer coordination entry points

This domain explains how Satoshium operates.

---

### 🧪 satoshium.xyz — Experimental Layer

Hosts simulations, research environments, and exploratory prototypes.

Provides:

- labs
- demos
- scenario experiments
- simulation surfaces

This domain explores what Satoshium may become.

---

### 🧰 satoshium.dev — Builder Layer

Supports developers working with Satoshium-aligned infrastructure.

Provides:

- SDK direction
- integration guidance
- build pathways
- contributor surfaces

This domain enables participation in the platform’s construction.

---

### 📚 satoshium.info — Knowledge Layer

Structured education and documentation environment.

Provides:

- learning materials
- technical explanations
- reference documentation
- long-form platform context

This domain explains how to understand Satoshium.

---

### 🔗 satoshium.link — Coordination Layer

Navigation and routing surface across the ecosystem.

Provides:

- repo map
- build progress tracking
- domain-layer mapping
- shared platform components

This domain connects the platform together.

---

### 🛍️ satoshium.store — Cultural Layer

Symbolic and material expression of the Satoshium ecosystem.

Provides:

- merchandise
- artifacts
- visual identity extensions

This domain reflects how Satoshium appears in the world.

---

## Why Domain Separation Matters

Separating platform layers across domains improves:

- architectural clarity
- long-term scalability
- governance flexibility
- contributor navigation
- public transparency

This structure allows Satoshium to grow as infrastructure rather than a single application surface.

---

Satoshium is being built as a layered coordination environment — not a monolithic system.
